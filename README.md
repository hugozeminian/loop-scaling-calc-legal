# loop-scaling-calc-legal

Public hosting for the privacy policy of **4-20mA Loop & PLC Calc** (iOS / Android).

`index.html` is the v1 privacy policy for the app. The app is local-first and offline: it
collects no personal data, requires no account, and has no backend. This repository exists only
to serve that policy at a public URL for the App Store and Google Play listings.

No application source code is hosted here.
